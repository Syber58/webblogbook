== Description ==

The plugin provided a shortcode [COVID19] to display Live Statistics, you can use the shortcode in posts or pages.

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Use shortcode [COVID19] to display Live Chart on the page or post.
